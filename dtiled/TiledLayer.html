    Struct TiledLayer

    A layer of tiles within the map.

    A Map layer could be one of: Tile Layer: data is an array of guids that each map to some tile from a TiledTileset Object Group: objects is a set of entities that are not necessarily tied to the grid Image Layer: This layer is a static image (e.g. a backdrop)

    Fields

    Name Type Description
    data uint[] An array of GIDs that identify tiles. Only for tilelayer
    draworder string Not documented by tiled, but may appear in JSON.
    height int Number of tile rows. Always same as map height in Tiled Qt.
    name string Name assigned to this layer
    objects TiledObject[] An array of objects. Only on objectgroup layers.
    opacity float Visual opacity of all tiles in this layer
    properties string[string] Optional user-defined key-value properties for this layer
    type TiledLayer.Type Category (tile, object, or image)
    visible bool whether layer is shown or hidden in editor
    width int Number of tile columns. Always same as map width in Tiled Qt.
    x int Horizontal layer offset. Always 0 in Tiled Qt.
    y int Vertical layer offset. Always 0 in Tiled Qt.

    Properties

    Name Type Description
    idxToCol [set] ulong get the column corresponding to a position in the data or objects array.
    idxToRow [set] ulong get the row corresponding to a position in the data or objects array.
