    Variable TiledTileset.tileproperties

    Optional per-tile properties, indexed by the GID as a string.

    For example, if the tile with GID 25 has the property "moveCost=4", then `tileproperties["25"]["moveCost"] == "4" A tile with no special properties will not have an index here. If no tiles have special properties, this field is not populated at all.

    Declaration

    struct TiledTileset
{
	// ...
	string[string][string] tileproperties;
	// ...
}
