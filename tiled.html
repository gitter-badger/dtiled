• tiled

    Module tiled

    Parse map files produced by Tiled. Currently only supports JSON format.

    Structs

    Name Description
    MapLayer A layer of tiles within the map.
    MapObject Represents an entity in an object layer.
    TiledMap Top-level Tiled structure - encapsulates all data in the map file.
    TileSet A `TileSet` maps GIDs (Global IDentifiers) to tiles.

    Enums

    Name Description
    TileFlag Flags set by Tiled in the guid field. Used to indicate mirroring and rotation.
